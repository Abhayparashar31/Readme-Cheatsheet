## README CHEATSHEET 

### 1. Headings 

To create a heding in Readme we need to use  #. Size of headings can be change by adding multiple  `#` Together. The greate the number of `#` is the smaller the heading is.
```
Symbol    Tag       Example
`#`       <h1>      # h1 heading
`##`      <h1>      # h2 heading
`###`     <h4>      # h3 heading
`####`    <h4>      # h4 heading
```
